üöÄ Local Test Server Setup

To properly test the widget without CORS issues, you need to serve the files from a local web server instead of opening them directly in the browser.

Method 1: Python (Recommended)

If you have Python installed:

Python 3:

cd /path/to/your/widget/folder
python -m http.server 8000

Python 2:

cd /path/to/your/widget/folder
python -m SimpleHTTPServer 8000

Then open: http://localhost:8000/test-dual-mode-widget.html

Method 2: Node.js

If you have Node.js installed:

npm install -g http-server
cd /path/to/your/widget/folder
http-server -p 8000

Then open: http://localhost:8000/test-dual-mode-widget.html

Method 3: PHP

If you have PHP installed:

cd /path/to/your/widget/folder
php -S localhost:8000

Then open: http://localhost:8000/test-dual-mode-widget.html

Method 4: Live Server (VS Code)

If you're using VS Code:

  1. Install the "Live Server" extension
  2. Right-click on test-dual-mode-widget.html
  3. Select "Open with Live Server"
