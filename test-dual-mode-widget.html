üöÄ How to Test

  1. Click the floating button in the bottom-right corner to open the widget
  2. Switch modes using the Voice Call / Text Chat buttons (if in dual mode)
  3. Test voice calls by clicking the "Call" button (works in demo mode)
  4. Test text chat by typing a message and pressing Enter or clicking Send
  5. Check responsiveness by resizing your browser window

üìã Configuration Options

  • data-mode: "voice", "chat", or "dual" (default: "dual")
  • data-default-mode: "voice" or "chat" (default: "voice")
  • data-voice-agent-id: Your Retell voice agent ID
  • data-chat-agent-id: Your Retell chat agent ID
  • data-webhook: Your Make.com webhook URL (required)
  • data-color: Primary color for the widget (default: "#347D9B")
  • data-title: Widget header title
  • data-subtitle: Widget header subtitle
