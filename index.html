Quick Start

Add the widget to any website with a single script tag:

<!-- Dual Mode (Voice + Chat) --> <script src="fluvio-universal-widget.js" data-webhook="https://hook.us2.make.com/your-webhook" data-project-id="your-project-id" data-mode="dual" data-default-mode="voice" data-title="AI Assistant" data-subtitle="Voice & Chat Support"></script> <!-- Voice Only --> <script src="fluvio-universal-widget.js" data-webhook="https://hook.us2.make.com/your-webhook" data-project-id="your-project-id" data-mode="voice"></script> <!-- Chat Only --> <script src="fluvio-universal-widget.js" data-webhook="https://hook.us2.make.com/your-webhook" data-project-id="your-project-id" data-mode="chat"></script>

Configuration Options

Required

  • data-webhook - Your Make.com webhook URL
  • data-project-id - Your Retell project ID

Mode Options

  • data-mode - "voice", "chat", or "dual"
  • data-default-mode - "voice" or "chat"

Customization

  • data-color - Primary color (hex)
  • data-title - Widget header title
  • data-subtitle - Widget header subtitle
  • data-position - Widget position

Dynamic Variables

  • data-company-name - Company name
  • data-agent-name - AI agent name
  • data-greeting - Custom greeting
